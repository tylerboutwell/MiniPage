## About

Minipage lets users create simple personal landing pages with links to social profiles, websites, and more.

---

## Features

- User authentication and profile management
- Add, edit, and reorder links
- Public profile pages (`/username`)
- Dark mode for the dashboard
- Mobile-responsive design
- Clean, modern UI with Tailwind CSS

---

## How to Use

1. Visit the live demo: [https://mysite-cpq9.onrender.com/](https://mysite-cpq9.onrender.com/)  
2. Sign up or log in to create your personal mini-page.  
3. Add your links and customize your profile.  
4. Share your public page at `https://mysite-cpq9.onrender.com/username`.
